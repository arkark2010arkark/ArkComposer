ArkComposer

A personal hobby project for AI-assisted composition

A personal hobby AI-driven Composer IDE with a built-in MCP server and a built-in AI that is still actively being trained.
Now with a full piano-roll style Edit Measure (MIDI) editor, direct AI Voicing and AI Accomp actions, a clearer top-bar workflow with status bar, and built-in update checking.
Connect Claude Code Desktop, Codex app, or any MCP-capable AI client and compose together in natural language.
개인적인 취미로 만드는 AI 기반 Composer IDE입니다. 피아노롤 스타일의 Edit Measure (MIDI) 편집기, AI Voicing·AI Accomp 직접 호출, 상태바가 포함된 새 상단 워크플로, 자동 업데이트 기능이 추가되었습니다. 악보 편집부터 AI 작곡까지 대화로 음악을 만들 수 있습니다.

Features  ·  기능

Everything you need
to compose.

작곡에 필요한 모든 것.

🎼
Full Score Editor
완전한 악보 편집기
Multi-track notation with notes, rests, dynamics, key/time signatures, lyrics, and chord symbols.

멀티트랙 악보 편집 — 음표, 쉼표, 나타냄말, 조표, 박자표, 가사, 화음 기호.
🎹
Edit Measure (MIDI)
마디 편집 — 피아노롤
Piano-roll style editor with Edit / Insert / Delete / Tie modes. Track-wide measure navigation, built-in playback, velocity shaping, and direct AI Voicing & AI Accomp actions — all without leaving the score workflow.

피아노롤 스타일 편집기 — 편집/삽입/삭제/타이 모드, 트랙 내 마디 탐색, 재생, 벨로시티 조절, AI Voicing·AI Accomp 직접 호출을 악보 워크플로 안에서 바로 사용.
🧭
Top-Bar Workflow & Status Bar
상단 바 워크플로 & 상태 바
Edit Measure is the primary action. Measure Select, Note Select, and Score Edit split selection from editing. A status bar shows current mode, tool, track, measures, rhythm, melody, and AI state at a glance.

Edit Measure가 주 동작. Measure Select·Note Select·Score Edit로 모드 분리. 상태 바에서 현재 모드, 도구, 트랙, 리듬, 멜로디, AI 상태를 한눈에 확인.
🤖
AI Voicing & AI Accompaniment
AI 보이싱 & AI 반주
AI Voicing from the top menu or inside Edit Measure (MIDI). AI Accomp generates accompaniment following melody and harmony. Styles include AI Chord Beat, AI Arpeggio, AI Free, AI Free (Melody Variation), and AI Drum Beat.

AI Voicing은 상단 메뉴 및 Edit Measure에서 직접 호출. AI Accomp는 멜로디·화성을 따르는 반주를 생성. AI Chord Beat, AI Arpeggio, AI Free, AI Free (Melody Variation), AI Drum Beat 스타일 지원.
🎵
Melody & Rhythm Variation
멜로디 & 리듬 변주
Inversion, retrograde, augmentation, pentatonic, passing tones. Genre-based rhythm patterns.

전위, 역행, 확대, 5음 음계, 경과음. 장르별 리듬 패턴 변주.
🎹
SoundFont Playback
사운드폰트 재생
Per-track instrument assignment with .sf2 SoundFont support. Enlarged volume controls, improved transport. Export to WAV and MP3 (higher quality).

트랙별 악기 지정, SF2 사운드폰트 지원. 개선된 볼륨 컨트롤·재생 흐름. WAV/MP3 고품질 내보내기.
📊
Melodic Mood Overlay
멜로딕 무드 오버레이
Visual stability/tension analysis on every note. 3 color styles: Jewel, Muted, Editorial.

음표별 안정/전환/긴장 시각화. 3가지 색상 스타일.
🔄
MIDI Import / Export
MIDI 가져오기 / 내보내기
Import MIDI files and convert to editable score. Export to standard MIDI format.

MIDI 파일을 악보로 변환. 표준 MIDI 형식으로 내보내기.
📑
Score Export
악보 내보내기
Export your score as PDF, PNG, or JPG. Native format: .arkscore / .json (human-readable).

PDF, PNG, JPG로 악보 내보내기. 자체 포맷 .arkscore / .json.
🎛️
Plugin Filter System
플러그인 필터 구조
Distortion, Echo, and more. Extensible plugin architecture for custom audio effects.

디스토션, 에코 등 오디오 이펙트 필터. 확장 가능한 플러그인 구조.
🔃
Auto Update Check
자동 업데이트 확인
Check for Updates compares your version against the latest GitHub Release and prepares the download/apply flow.

Check for Updates로 최신 GitHub Release와 버전을 비교하고 다운로드/적용 흐름을 안내합니다.
AI Integration  ·  AI 연동

Talk to your score.

악보에게 말을 거세요.

✅ Tested Mode 1 — stdio · Claude Code Desktop
Launch ArkComposer as an MCP stdio server and connect directly from Claude Code Desktop.
Claude Code Desktop에서 직접 MCP stdio로 연결합니다.
// claude_desktop_config.json { "mcpServers": { "arkcomposer": { "command": "ArkComposer.exe", "args": ["--mcp-mode"] } } }
✅ Tested Mode 2 — HTTP/SSE · Codex App
Start the MCP server from the ArkComposer UI, then register the HTTP/SSE endpoint in the Codex app MCP settings.
ArkComposer에서 MCP 서버를 켠 뒤, Codex app MCP 설정에 HTTP/SSE 주소를 등록합니다.
# 1. Run ArkComposer normally # 2. Left palette → MCP Server → Start # 3. Open Codex app → Settings → MCP servers # 4. Set URL to: http://localhost:9101/sse
✅ Tested Mode 3 — TCP · Advanced / CLI
For raw JSON-RPC clients, use the TCP port exposed by ArkComposer. This is useful for Codex CLI or custom MCP tooling.
raw JSON-RPC 클라이언트는 ArkComposer의 TCP 포트에 직접 연결할 수 있습니다. Codex CLI나 커스텀 도구에 적합합니다.
# 1. Run ArkComposer normally # 2. Left palette → MCP Server → Start # 3. Connect your client to: localhost:9100
Note for Codex app: If you want to switch between HTTP/SSE and stdio inside the Codex app, uninstall the existing ArkComposer MCP entry first, then create the new one.
Codex app에서는 HTTP/SSE와 stdio 타입을 서로 바꾸려면, 기존 ArkComposer MCP 항목을 먼저 제거한 뒤 새로 등록해야 합니다.

Note on ChatGPT: ChatGPT does not natively support MCP. A separate OpenAPI server is required to bridge ChatGPT function calls to ArkComposer's MCP tools.
ChatGPT는 MCP를 직접 지원하지 않습니다. ChatGPT 연동을 위해서는 별도의 OpenAPI 서버 구현이 필요합니다.
Claude Code Desktop Demo  ·  실제 연동 화면

Claude Code Desktop — ArkComposer MCP Connected  ·  실제 연동 화면

Claude Code Desktop

ArkComposer as MCP Server · MCP 서버로 연동

  • Add ArkComposer to claude_desktop_config.json as an MCP server using --mcp-mode claude_desktop_config.json에 MCP 서버로 등록합니다.
  • Open Claude Code Desktop → Click Connector → Enable arkcomposer toggle Claude Code Desktop 실행 → 커넥터 → arkcomposer 활성화
  • Start composing in natural language: "Create a 8-bar melody in C major" 자연어로 작곡 시작: "C장조로 8마디 멜로디 만들어줘"
  • ArkComposer updates the score in real time as the AI edits it AI가 악보를 편집하는 동시에 ArkComposer에서 실시간 반영됩니다.
Codex App Demo  ·  실제 연동 화면

Codex App

Fast public setup for ArkComposer MCP · 일반 사용자용 빠른 설정

Need a full walkthrough with larger screenshots? Open the dedicated Codex App Guide.
큰 화면 이미지와 자세한 설명은 별도 Codex App Guide 페이지에서 볼 수 있습니다.

  • Open ArkComposer normally, then click Left Palette → MCP Server and start the server. ArkComposer를 일반 실행한 뒤, 왼쪽 팔레트 → MCP Server를 눌러 서버를 시작합니다.
  • In the Codex app, open Settings → MCP servers and register the HTTP/SSE endpoint http://localhost:9101/sse. Codex app에서 Settings → MCP servers로 이동해 http://localhost:9101/sse 주소를 등록합니다.
  • If you prefer stdio instead, set ArkComposer.exe with argument --mcp-mode. stdio 방식을 원하면 ArkComposer.exe와 --mcp-mode를 등록합니다.
  • Start with direct prompts such as “Compose a ballad song with 20 measures.” or “Add a 4-bar intro to the current song.” 예: “20마디 발라드 곡 만들어줘”, “현재 곡에 4마디 전주 넣어줘”처럼 바로 지시하면 됩니다.
  • Codex reads the current score through MCP, plans the edit, and ArkComposer reflects the result in real time. Codex가 MCP로 현재 악보를 읽고 편집 계획을 세우면, 결과가 ArkComposer에 실시간으로 반영됩니다.

Live composing flow: Codex prompt on the left, ArkComposer score updating on the right. This image works best as a large overview, not as a small thumbnail.

Step 1 in ArkComposer: open the left palette, click MCP Server, and start the server. The default ports are TCP 9100 and HTTP/SSE 9101.

Recommended in the Codex app: register the MCP server URL as http://localhost:9101/sse. This is the simplest path for most users.

Alternative setup: use stdio with ArkComposer.exe and the argument --mcp-mode if you want the client to launch ArkComposer directly.

The conversation itself should stay simple: connect first, then ask for concrete music tasks in plain language.

Built-in AI Accompaniment  ·  내장 AI 반주

Built-in AI Accompaniment  ·  내장 AI 반주 생성 화면

Built-in AI

AI Voicing · AI Accomp · Local ONNX Models  ·  로컬 AI 모델

  • Access AI Voicing from the AI Helper menu or directly inside Edit Measure (MIDI) for cleaner voicing and redistribution. AI Helper 메뉴 또는 Edit Measure (MIDI) 안에서 AI Voicing을 바로 호출하여 보이싱을 개선합니다.
  • Use AI Accomp inside Edit Measure (MIDI) to generate accompaniment that follows the current melody and harmony. Edit Measure (MIDI) 안에서 AI Accomp를 사용해 현재 멜로디·화성에 맞는 반주를 생성합니다.
  • Choose from styles: AI Chord Beat, AI Arpeggio, AI Free, AI Free (Melody Variation), AI Drum Beat AI Chord Beat, AI Arpeggio, AI Free, AI Free (Melody Variation), AI Drum Beat 등 다양한 스타일.
  • Built-in ONNX and TinyGRU models run locally — no internet required. Models improve as training progresses. 내장 ONNX 및 TinyGRU 모델은 로컬에서 동작하며 인터넷 없이 사용 가능. 학습에 따라 지속 개선.
