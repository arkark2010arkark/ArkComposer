ArkComposer Release Notes

Version 1.0.1.1
Date: 2026-03-21

Summary
- Reworked note spacing, tied-segment handling, and dense-measure readability across score rendering.
- Normalized unnecessary split/tied note representations on JSON/MIDI load and reduced differences between loaded data and pasted data.
- Improved accompaniment generation and top chord display by switching harmony inference from a single-track reference to a merged harmony reference.
- Cleaned up major GUI workflows including the left palette, player controls, and edit/selection modes.

Notation / Layout
- Unified displayed note X positions and hit-test X positions under the same spacing logic.
- Improved adaptive spacing using measure density, note duration, and visible symbol count including rests.
- In sparse measures, spacing now follows duration proportion more strongly while still guaranteeing a minimum width for short notes such as 32nd notes.
- Reduced cases where tied fragments consumed excessive horizontal space by rebuilding spacing from untied temporary notation.
- Separated tie rendering responsibilities from spacing responsibilities so tie graphics no longer inflate spacing by themselves.
- Relaxed automatic notehead shrinking so it is used only when local gaps are actually too tight.
- Added green insertion guide lines in edit mode for valid note start positions inside rest windows.

Import / Normalize
- Added canonical notation duration cleanup for JSON/MIDI import when values are close to standard notation lengths.
- Added automatic merge of reducible tied segments during Normalize when no new onset needs to be shown.
- Changed app JSON loading so stored micro-rest segments are not reinserted directly; notes are restored first and rests are rebuilt by Normalize.
- Reduced mismatches between score notation immediately after load and after copy/paste round-trips.

Harmony / Accompaniment / Chords
- Added a song-level harmony reference track concept with save/load support.
- Unified accompaniment generation, AI accompaniment context, and top chord symbol display around a shared reference path.
- Harmony inference now prefers a temporary merged harmony reference built from all non-percussion tracks per measure.
- Excluded the target accompaniment track from the merged harmony source to avoid self-reference during accompaniment generation.

Editing / Interaction
- Added eraser tools.
  - Erase Measure: clears the clicked measure while keeping the measure slot.
  - Erase Note: deletes only the clicked note.
- Simplified the top edit mode structure to two modes:
  - Measure Edit
  - Note Edit
- Measure selection is now available only in Measure Edit mode.
- Note selection is now available only in Note Edit mode.
- Removed right-click quick selection on the score.
- Removed long-press mouse pitch dragging for notes. Keyboard note movement remains available.
- When playback requires a selected measure range and no measure is selected, playback now falls back to the measure containing the last worked note.
- Playback no longer forces the UI to switch modes while playing.

Player / Palette / UI
- Added three playback start modes:
  - Selected range only
  - From selected range to end
  - Whole song
- Changed Play after Pause to resume from the current position instead of restarting from the beginning.
- Changed Ctrl+mouse-wheel zoom to follow the measure under the mouse cursor using score-space anchors, and kept the hovered measure's left boundary visible during reflow so zooming does not scroll past the measure start.
- Increased player control area and slider hit areas for easier volume and transport control.
- Increased the default startup window size and lowered the default score zoom so more measures are visible immediately after launch.
- Reorganized the left palette into these top-level sections:
  - File
  - Edit
  - Score Setup
  - Input
  - Track
  - Compose Assist
- Grouped Notes, Rests, Erasers, and Accidentals inside a single Input section, with rests reduced to a secondary row.
- Replaced the eraser icons with clearer large/small eraser visuals.
- Replaced single-track delete selection with a multi-select checklist popup so multiple tracks can be deleted at once.

Localization / Docs
- Added missing Korean localization keys in accompaniment-related menus that previously fell back to English.
- Updated help text and developer documentation to reflect the new player modes, palette structure, eraser tools, and two-mode editing contract.
