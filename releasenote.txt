ArkComposer Release Notes

Version 1.1.1.2
Date: 2026-04-05
Base commit: 982082d

Summary
- Expanded `Edit Measure (MIDI)` into a stronger piano-roll style measure editor with direct note repair, tie editing, velocity editing, and built-in AI actions.
- Reorganized the top workflow around `Edit Measure`, `Measure Select`, `Note Select`, and `Score Edit`, with clearer grouping and a new status bar.
- Improved AI composition flows including `AI Voicing`, `AI Accomp`, new AI accompaniment styles, and safer model fallback behavior.
- Added built-in update checking for GitHub Releases and refreshed the bundled help and README to match the current UI.

Edit Measure (MIDI)
- Upgraded the dialog from a simple duration-adjust window into a broader MIDI-style measure editor.
- Added dedicated Edit / Insert / Delete / Tie modes inside the measure editor.
- Added track-wide measure navigation so nearby measures can be opened and checked more quickly.
- Added direct playback controls in the editor for faster audition while editing.
- Added `Velocity`, `AI Voicing`, and `AI Accomp` actions directly inside the editor.
- Fixed tie information loss during save and improved stability around measure-edit workflows.
- Improved the loading/wait flow when playback is triggered from the measure editor.

Top Bar / UI / Workflow
- Promoted `Edit Measure` to the primary top-bar action.
- Split top-bar work modes into `Measure Select`, `Note Select`, and `Score Edit`.
- Restored the workflow where pressing a pinned note or eraser tool automatically switches into score editing.
- Added a top status bar that shows current mode, tool, track, selected measures, rhythm, melody, and AI state.
- Reorganized composition actions under Rhythm, Melody, Helper, and AI Helper menus.
- Added a settings dialog and refreshed several menu labels and help strings to match the current screen layout.

AI / Compose Assist
- Added direct `AI Voicing` access from the top menu and from `Edit Measure (MIDI)`.
- Added direct `AI Accomp` access from `Edit Measure (MIDI)`.
- Added new AI accompaniment styles including `AI Free (Melody Variation)`.
- Recovered and improved `AI Free` accompaniment behavior.
- Fixed a bug where accompaniment generation style could stay stuck on the AI-recommended entry.
- Improved model fallback so missing newer phases can fall back more safely to earlier available models.
- Continued improving hybrid AI usage where accompaniment and voicing can use different model generations more reliably.

Playback / Player
- Enlarged the player volume control and slider hit area so volume changes are easier to perform.
- Improved transport behavior and loading feedback around playback launched from advanced editing flows.
- Raised MP3 export quality.

Fixes
- Fixed several bugs in rhythm reassignment, including cases where other measures could be removed unexpectedly.
- Improved AI drum post-processing and related generation handling.
- Fixed localization gaps so newer AI and measure-edit commands no longer fall back to the wrong language as often.
- Updated built-in help content to reflect the current top bar, status bar, measure-edit workflow, and AI tools.

Update / Distribution
- Added `Check for Updates` support based on GitHub Releases.
- The app can compare the running version with the latest published release and prepare a download/apply flow for packaged updates.
